一款非常简单的新闻类app，通过刷新来获取一些随机新闻
#手刷新闻 每日为您推送全国各类趣闻信息，趣闻段子，生活趣事，明星娱乐，国内外最新的娱乐信息，无论你是星迷 还是科技迷...让你足不出户 就能了解到国内外各行各业,各领域方面的最新消息，赶快下载体验吧...
## 一、项目技术框架
1. 应用数据：聚合数据<br>
2. 图片加载：glide
3. 下拉刷新：谷歌的SwipeRefreshLayout控件
4. 采用easyrecyclerview控件展示新闻列表
